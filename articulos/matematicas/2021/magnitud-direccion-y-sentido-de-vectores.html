INTRODUCCIÓN

Cuando se comienza a estudiar Álgebra Lineal, se comienza a manejar constantemente el concepto de vector. Un vector lo podemos definir como una recta con dirección. Normalmente en álgebra lineal, un vector comienza desde el origen de un plano cartesiano, que es simplemente la coordenada (0, 0).

En la imagen anterior, el vector "a" se muestra que comienza desde el origen.

Al comenzar a manejar vectores, se deben conocer ciertas características de estos mismos, que es precisamente de las que va a tratar este artículo:

DIRECCIÓN DE UN VECTOR

La dirección de un vector se define como el ángulo forma con una línea horizontal. Por lo general se toma al eje x como esta línea, pero no esta limitado a este.

SENTIDO DE UN VECTOR

El sentido de un vector se refiere hacia donde apunta el mismo. Particularme se usan como referencia las 4 secciones de un plano cartesiano, definidas por los ejes "y" y "x", junto con sus zonas positivas y negativas.
